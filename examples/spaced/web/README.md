# web

web-based visualization for Spaced CLI.

## Service surfaces

The browser router includes sample adjacent service surfaces for demos:

- `service list` — REST, WebSocket, MCP, and event stream surfaces.
- `service inspect websocket` — socket paths and telemetry frame samples.
- `service inspect mcp` — MCP tool/resource names and sample payloads.
- `service smoke` — verifies each sample surface has endpoint payloads.
