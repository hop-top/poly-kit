## Summary

One-command reproducible dev environment via nix, Docker, or Codespaces.
Nix defines all toolchains; multi-stage Dockerfile packages them into a
slim runtime. AI coding tools installed on-demand, never baked in.

## Acceptance criteria

- [ ] `nix develop` enters shell with Go, Node, Python, all linters
- [ ] `devbox shell` produces identical environment
- [ ] VS Code Dev Containers opens with all extensions configured
- [ ] GitHub Codespaces uses same devcontainer.json
- [ ] `task ai:setup` offers interactive tool selection
- [ ] `.ai-tools` file triggers auto-install on container create
- [ ] CI reuses same Dockerfile via composite action
- [ ] Container starts without nix runtime (instant startup)
